Docker
│
├── Immagini (Images)
│   ├── docker build → crea un'immagine dal Dockerfile
│   ├── docker pull  → scarica un'immagine dal repository
│   └── docker rmi   → cancella un'immagine locale
│
└── Contenitori (Containers)
    ├── docker run   → avvia un contenitore da un'immagine
    ├── docker ps    → mostra i contenitori attivi
    ├── docker stop  → ferma un contenitore in esecuzione
    ├── docker rm    → rimuove un contenitore fermo
    └── docker push  → carica un'immagine su un repository remoto
